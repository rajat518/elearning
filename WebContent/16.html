Grouping

Grouping CSS selectors allows you to put the same styles on many different elements without repeating the styles in your style sheet. This means that your style sheet will be smaller and download more quickly.
Separate multiple grouped selectors in the style property by commas. In this example the style will affect the p and div elements: p, div { color: #f00; } Any form of selector can be grouped with any other selector. In this example, a class selector is grouped with an ID selector:
