Pseudo Classes

CSS pseudo-classes are used to add special effects to some selectors. You do not need to use Javascript or any other script to use those effects. A simple syntax of pseudo-classes is as follows: selector:pseudo-class {property: value} CSS classes can also be used with pseudo-classes: selector.class:pseudo-class {property: value}
